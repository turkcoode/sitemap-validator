Sitemap Validator

XML sitemap dosyalarını doğrulayan ve analiz eden CLI aracı

Hızlı Başlangıç

npm install -g sitemap-validator

sitemap-validator --help

Özellikler

Kolay Kullanım

Tek komutla kurulum ve çalıştırma. Sıfır yapılandırma gerektirir.

Hızlı ve Hafif

Minimum bağımlılık ile yüksek performans sunar.

CLI ve API

Hem komut satırından hem de programatik olarak kullanılabilir.

Türkçe Destek

Tam Türkçe karakter ve dil desteği ile geliştirilmiştir.

Kullanım Örneği

const sitemapvalidator = require('sitemap-validator');

const result = sitemapvalidator.analyze(input);
console.log(result);
