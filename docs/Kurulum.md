# Kurulum Rehberi

## Gereksinimler

- **Node.js** >= 14.0.0
- **npm** >= 6.0.0

## Global Kurulum (CLI)

Komut satırından doğrudan kullanmak için global olarak kurun:

```bash
npm install -g sitemap-validator
```

## Proje Bağımlılığı

Projenize bağımlılık olarak eklemek için:

```bash
npm install sitemap-validator
```

Veya yarn ile:

```bash
yarn add sitemap-validator
```

## Doğrulama

Kurulumun başarılı olduğunu doğrulayın:

```bash
sitemap-validator --version
```

## Güncelleme

```bash
npm update -g sitemap-validator
```

## Sorun Giderme

Kurulum sırasında sorun yaşarsanız:

1. Node.js sürümünüzü kontrol edin: `node --version`
2. npm cache temizleyin: `npm cache clean --force`
3. Tekrar deneyin: `npm install -g sitemap-validator`

Hâlâ sorun yaşıyorsanız [GitHub Issues](https://github.com/turkcoode/sitemap-validator/issues) üzerinden bildirebilirsiniz.

Detaylı kurulum rehberi için [TurkCode](https://turkcode.net/blog) blogunu ziyaret edin.
